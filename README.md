# 🎮 Koji Adventure: $ANIMAL Edition

![Game Status](https://img.shields.io/badge/Status-Live-brightgreen?style=for-the-badge)
![Web3 Style](https://img.shields.io/badge/Theme-Web3--Community-gold?style=for-the-badge)
![Platform](https://img.shields.io/badge/Platform-Mobile--Web-blue?style=for-the-badge)

**Koji Adventure** adalah sebuah permainan *platformer climber* yang dirancang khusus untuk komunitas Web3. Pemain berperan sebagai **Koji** yang harus melompat dari satu platform ke platform lainnya untuk mengumpulkan token **$ANIMAL** sambil bertahan hidup dari serangan musuh.

---

## 🚀 Fitur Unggulan

* **Double Jump Mechanic**: Memungkinkan pemain melompat lebih tinggi dengan menekan tombol lompat dua kali.
* **Dynamic Moving Platforms**: Beberapa lantai bergerak secara horizontal, menuntut ketepatan waktu dalam melompat.
* **Sticky Physics**: Karakter Koji akan otomatis menempel dan bergerak mengikuti lantai batu yang berjalan.
* **$ANIMAL Collection**: Sistem skor berbasis token yang relevan dengan ekosistem proyek Anda.
* **Optimized Mobile Controls**: Tata letak tombol D-Pad segitiga di pojok kanan bawah yang intuitif dan tidak menghalangi pandangan.
* **Recovery System**: Munculnya **Ramuan Ajaib** secara berkala untuk memulihkan nyawa pemain.

---

## 🛠️ Detail Teknis

Game ini dibangun menggunakan teknologi web murni tanpa *library* eksternal untuk memastikan kecepatan muat yang maksimal:
* **Language**: JavaScript (ES6+)
* **Rendering**: HTML5 Canvas API
* **Styling**: CSS3 dengan sistem Grid & Flexbox
* **Assets**: Custom Sprite Images (`.png`)

---

## 🎮 Instruksi Bermain

1.  **Cara Menjalankan**: Cukup buka file `index.html` pada browser smartphone atau komputer Anda.
2.  **Kontrol**:
    * **Kiri (◀)**: Bergerak ke arah kiri.
    * **Kanan (▶)**: Bergerak ke arah kanan.
    * **Atas (▲)**: Melompat. Tekan lagi saat di udara untuk *Double Jump*.
3.  **Misi**: Kumpulkan token sebanyak mungkin dan capai titik tertinggi tanpa kehilangan 3 nyawa.
